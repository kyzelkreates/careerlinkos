# CareerLink OS™
**Job Search Compliance Dashboard + Jobseeker Activity PWA**
*Powered by 4P3X Intelligent AI — Created by Kyzel Kreates*

---

## Overview
CareerLink OS™ is a local-first employment support platform for coaches and caseworkers.
It tracks jobseeker activity, weekly job-search targets, applications, interviews, check-ins, evidence, and support risks.

## Products
- **CareerLink Coach Dashboard** — Full employment support management interface for coaches/advisors
- **CareerLink Jobseeker PWA** — Mobile-first progressive web app for jobseekers to log activity on the go

## Key Features
- Weekly job-search target tracking (default: 35 hours, configurable)
- Activity logging with 12 activity types
- Application and interview tracking
- Daily check-ins with 10 questions
- Evidence record management
- Support flags and barrier tracking
- AI risk insights (4P3X CareerLink Intelligence Layer™)
- Demo mode for presentation and training
- PWA installable on mobile devices
- Offline-first local data storage

## AI Layer
CareerLink OS™ uses the **4P3X CareerLink Intelligence Layer™** in LOCAL / rule-based mode by default.
No API key is required. The AI provides advisory insights only and does not replace professional judgement.

**AI Disclaimer:** CareerLink AI provides guidance, organisation support, and dashboard help only.
It does not replace official guidance, legal advice, medical advice, benefits advice, or human coach/caseworker review.

## Getting Started
1. Open the app — you will be guided through first-run setup
2. Enable Demo Mode in Settings to see sample data
3. Use Settings → Invite Jobseeker to generate the PWA link for your jobseekers
4. Add real jobseekers and activity data when ready

## Tech Stack
- React 18 + Vite
- Zustand (local-first state)
- Tailwind CSS
- PWA (Workbox)
- localStorage SSOT (no backend required)
